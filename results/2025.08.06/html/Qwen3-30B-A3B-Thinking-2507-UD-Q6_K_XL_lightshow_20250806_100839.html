Technical Note: This is a pure WebGL implementation using ray marching with Signed Distance Functions (SDFs). The metaballs use smooth minimum blending for organic merging, and the color palette cycles through psychedelic hues using time-based HSV modulation. All animation is driven by the time uniform updated via requestAnimationFrame.
