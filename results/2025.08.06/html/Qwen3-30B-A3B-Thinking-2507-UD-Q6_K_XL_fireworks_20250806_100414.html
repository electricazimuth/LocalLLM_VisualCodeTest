Autonomous fireworks display - no user interaction needed. The simulation automatically launches fireworks from the bottom of the canvas.

Each firework travels upward, explodes into colorful particles, and falls realistically under gravity.

Implementation Notes

Object-Oriented Design: Separate Rocket and ExplosionParticle classes with distinct properties and behaviors

Efficient Management: Single particles array with optimized removal using filter() and batch processing

Physics Simulation: Realistic gravity (1.0 px/sÂ²), velocity-based movement, and lifespan fading

Performance: Delta-time based animation for consistent speed across frame rates

Autonomous Launching: Random intervals (500-2000ms) with smooth transition between fireworks

Visual Design: Dark background (black) with vibrant colors, particles fade out naturally

Optimization: Avoids DOM manipulation in the game loop, uses canvas for direct rendering
