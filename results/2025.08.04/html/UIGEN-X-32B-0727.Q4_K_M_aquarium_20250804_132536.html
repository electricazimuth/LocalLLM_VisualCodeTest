About This Simulation

This aquarium simulation demonstrates JavaScript best practices for creating autonomous agents with realistic behaviors.

Key features include:

  • Object-oriented design with Fish class
  • Smooth animation using requestAnimationFrame
  • Frame-rate independent movement with delta time
  • Natural swimming patterns with wandering behavior
  • Realistic boundary interactions
  • Dynamic lighting effects
