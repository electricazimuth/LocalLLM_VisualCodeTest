Developer notes regarding collision detection method, state management, design choices, etc.
  • Collision detection uses simple rectangle intersection.
  • State management is handled using a string variable.
  • The explosion effect is created by generating a burst of particles with random velocities and lifespans.
  • Delta time is used for smooth animation.
  • Wrapping is implemented by checking if a vehicle goes beyond the canvas boundaries and resetting its position to the opposite edge.
