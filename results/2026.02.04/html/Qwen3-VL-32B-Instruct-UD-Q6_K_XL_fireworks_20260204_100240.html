Notes:

This implementation uses a single Particle class with a type property to distinguish between rockets and explosion particles, enabling efficient management in one array.

Delta time is calculated using timestamp - lastTime for frame-rate independent animation.

Rockets are launched autonomously at random intervals (0.5 to 2 seconds) from the bottom-center of the canvas.

Each rocket explodes after 1.5 seconds or when it reaches 20% of the canvas height, creating 30-80 explosion particles with random directions and lifespans (1-3 seconds).

Explosion particles are drawn with alpha fading based on their remaining lifespan for a realistic fade-out effect.

Particles are removed efficiently by iterating backwards through the array and using splice to avoid index issues.

Gravity is applied to all particles in the update loop, making explosion particles arc realistically.

The canvas is cleared before each frame to prevent ghosting.

Colors are randomized using HSL for vibrant, consistent fireworks.

Performance is optimized by minimizing DOM access, using a single particle array, and efficient removal.
